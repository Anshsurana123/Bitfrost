# Bitfrost 🌈
> **Unified Repository for BIFRÖST B2B AI Gateway**

This monorepo compiles all components of the Bifröst B2B AI Gateway project:

1. **`backend/`**: Sovereign proxy written in Go acting as a secure, high-performance, and cost-optimizing gateway between client devices and LLM APIs.
2. **`dashboard/`**: Next.js dashboard providing real-time latency graphs, cost savings counters, key vault manager, and security logs.
3. **`gateway/`**: Documentation, database schema setups, and example environmental files (formerly `bifrost-showcase`).

Refer to the individual READMEs in each folder for specific setup and configuration instructions.
